# Asistencia Portfolio

> **Aviso:** Este proyecto ha sido creado con **fines demostrativos** para mostrar habilidades técnicas en Angular, PrimeNG, y desarrollo de componentes web. No está destinado para uso en producción.

## Acerca del Proyecto

Aplicación web para la gestión de asistencia académica. Permite registrar y consultar la asistencia de estudiantes, así como generar reportes individuales y por curso.

## Tecnologías

- Angular 17+
- PrimeNG 17
- TypeScript
- RxJS
- Karma / Jasmine (testing)

## Funcionalidades

- **Listar Asistencia:** Visualización de estudiantes con filtros en cascada (periodo, modalidad, curso, paralelo, asignatura), reportes PDF y diálogo informativo.
- **Registrar Asistencia:** Creación y actualización de asistencia diaria por clase, con validación de fecha y control de faltas por estudiante.
